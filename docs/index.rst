Jawaf
=================================

**Jawaf asynchronous web application framework**

A fast and modular asynchronous web application framework built on Python 3.6+ and Sanic.

It aims to wrap Sanic in a Django like fashion - providing database connectivity, sessions, management commands and other essentials to building a modern web stack baked in.
